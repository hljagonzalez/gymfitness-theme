# Gym Fitness — Tema WordPress

Tema personalizado para el sitio GymFitness (desarrollo local con [Local](https://localwp.com/)).

## Instalación

1. Copia esta carpeta en `wp-content/themes/gymfitness/`.
2. Activa el tema en **Apariencia → Temas**.
3. Si usas ACF, importa o sincroniza los campos desde `acf-json/`.

## Respaldo del sitio completo

El contenido (entradas, medios, base de datos) no está en este repositorio. Usa **Export site** en Local o tu método de backup habitual.
